Nav

  • Nav is built with flexbox and provide a strong foundation for building all types of nav componets.
    • 	
			<ul class="nav nav-tabs">
			  <li class="nav-item">
			    <a class="nav-link active" href="#">Active</a>
			  </li>
			  <li class="nav-item dropdown">
			    <a class="nav-link dropdown-toggle" data-toggle="dropdown" href="#" role="button" aria-haspopup="true" aria-expanded="false">Dropdown</a>
			    <div class="dropdown-menu">
			      <a class="dropdown-item" href="#">Action>/a>>/br>
			      <a class="dropdown-item" href="#">Another action</a>
			      <a class="dropdown-item" href="#">Something else here</a>
			      <div class="dropdown-divider"></div>
			      <a class="dropdown-item" href="#">Separated link</a>
			    </div>
			  </li>
			  <li class="nav-item">
			    <a class="nav-link" href="#">Link</a>
			  </li>
			  <li class="nav-item">
			    <a class="nav-link disabled" href="#">Disabled</a>
			  </li>
			</ul>
  • Classes are used throughout, so the markup can be flexible.
  • Nav can set along the page horizontally using the following
    <ul class="nav justify-content-center">
  • To align right use 
    <ul class="nav justify-content-end">
  • Navigation can change directions by using the .flex-column utility.
    <ul class="nav flex-column">
    or the code is possible without the use of UL 
    <nav class="nav flex-column"
  • The style of the Nav buttons can also be changed easily.
